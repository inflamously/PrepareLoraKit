App Connection Map

The central join is the step type string. Project YAML uses a type such as CaptionBboxStep; STEP_TYPE_MAP parses its config, and STEP_INVOKE_MAP chooses the adapter that runs it.

Control And Data Flow

LayerWhat to readWhy it matters
Entrypointsmain.py, prepare_lora_kit/cli/Registers plk commands and starts CLI or UI runs.
Frontendprepare_lora_kit_ui/static/app.js, prepare_lora_kit_ui/static/core/api.jsBoots the static UI and calls the pywebview bridge.
UI bridgeprepare_lora_kit_ui/bridge.pyExposes Python methods to window.pywebview.api.
UI runnerprepare_lora_kit_ui/runner/manager.pyAdds job status, step selection, interactions, cancellation, and logs.
Configprepare_lora_kit/project/base.py, prepare_lora_kit_pipeline/configuration.pyLoads project YAML, validates order, and maps step types to config classes.
Dispatchprepare_lora_kit/pipeline.py, prepare_lora_kit/invoke/__init__.pyWalks the configured pipeline and calls the right step adapter.
Workprepare_lora_kit/steps/import_step through export_stepImplements the image preparation stages.
State/outputprepare_lora_kit/utils/state.py, outputs/name/Tracks completed steps and stores dataset, reports, and optional export artifacts.

Read This First

  1. README.md for the user workflow and output layout.
  2. prepare_lora_kit/pipeline.py for the core CLI run loop.
  3. prepare_lora_kit/invoke/__init__.py for step type to adapter dispatch.
  4. prepare_lora_kit/project/base.py for project YAML to ProjectConfig.
  5. prepare_lora_kit_ui/bridge.py, then prepare_lora_kit_ui/runner/manager.py for UI behavior.
